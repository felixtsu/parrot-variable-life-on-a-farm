## 用作扩展

此仓库可以作为 **插件** 添加到 MakeCode 中。

* 打开 [https://arcade.makecode.com/](https://arcade.makecode.com/)
* 点击 **新项目**
* 点击齿轮图标菜单下的 **扩展**
* 搜索 **https://github.com/felixtsu/arcade-woodcutter-and-axe** 并导入

## 编辑此项目 ![构建状态标志](https://github.com/felixtsu/arcade-woodcutter-and-axe/workflows/MakeCode/badge.svg)

在 MakeCode 中编辑此仓库。

* 打开 [https://arcade.makecode.com/](https://arcade.makecode.com/)
* 点击 **导入**，然后点击 **导入 URL**
* 粘贴 **https://github.com/felixtsu/arcade-woodcutter-and-axe** 并点击导入
